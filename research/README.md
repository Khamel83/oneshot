# ONE_SHOT Research Archive

Deep research outputs saved here by the deep-research skill.

## Format

`<topic-slug>/research.md` - Full research report with:
- Executive Summary
- Overview & Core Concepts
- Current State
- Technical Deep Dive
- Practical Applications
- Challenges & Open Problems
- Future Outlook
- Resources

## Search

To find past research:
```bash
grep -r "Executive Summary" ~/github/oneshot/research/
```
