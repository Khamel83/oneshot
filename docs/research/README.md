# Research Archives

This directory contains research results from Gemini CLI free-search sessions.

## File Naming

- `{YYYY-MM-DD}_{topic_slug}_in_progress.md` - Live research updates
- `{YYYY-MM-DD}_{topic_slug}_final.md` - Final polished reports

## Usage

From Gemini CLI:
```
/free-search What are the latest developments in quantum computing?
```

From Claude Code:
```
Use free-search to find information about X
```

## Auto-Created

All files are automatically created by the `/free-search` command.
